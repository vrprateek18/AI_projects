# Placement Predictor

## Overview

This Machine Learning project predicts whether a student will get placed based on:

* CGPA
* IQ Score

The project uses Logistic Regression to classify students as:

* Placed (1)
* Not Placed (0)

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-Learn
* Mlxtend

## Project Workflow

1. Load Dataset
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Feature Scaling
6. Train Logistic Regression Model
7. Make Predictions
8. Evaluate Model Performance
9. Visualize Decision Boundary

## Files

* placement-predictor1.ipynb — Main notebook
* requirements.txt — Project dependencies

## Installation

Install required packages:

pip install -r requirements.txt

* Logistic Regression
